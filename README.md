# Simodos Europe eSIM landing page

Responsive standalone landing page for the Europe 400 GB / 30 days / $50 promotion.

## Files

- `index.html` — page structure and sales copy
- `styles.css` — responsive Simodos styling
- `script.js` — mobile menu, FAQ accordion and subtle reveal effects
- `assets/` — optimized logo and original generated images

## Publish

Upload the complete folder to the desired public directory on the Simodos website. Keep the file and folder names unchanged so the asset paths continue to work.

All purchase buttons currently open:

`https://appm.simodos.net/`

If a direct product checkout URL becomes available, replace that URL in `index.html` with the direct link.

## Important product text included

- 400 GB high-speed data
- Additional 100 GB bonus data
- 30-day validity starts immediately after purchase
- Supported European coverage list
- Data only; smartphones only
- eSIM-enabled iOS and Android devices
- QR-code activation, 4G/5G, hotspot and renewal
- APN: `netmon.vodafone.it`
- Plan code: `220110`
